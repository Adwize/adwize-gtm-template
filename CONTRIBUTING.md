# Contributing to Adwize GTM Template

Thanks for your interest in contributing! Here's how to get started.

## How to Contribute

1. **Fork** the repository
2. **Create a branch** from `main` (`git checkout -b feature/my-change`)
3. **Make your changes** and ensure the GTM template tests pass
4. **Commit** with a clear message describing what you changed and why
5. **Open a Pull Request** against `main`

## Guidelines

- Keep changes focused — one feature or fix per PR
- Do not commit secrets, API keys, or credentials
- Follow the existing code style in `template.tpl`
- Update `README.md` if your change affects configuration or usage

## Reporting Issues

Open a [GitHub Issue](https://github.com/Adwize/adwize-gtm-template/issues) with:

- A clear title and description
- Steps to reproduce (if applicable)
- Expected vs. actual behavior

## Code of Conduct

Be respectful and constructive. We reserve the right to remove comments or block contributors who are disruptive.
